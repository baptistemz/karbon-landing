L'application

Scannez le code barre de votre produit alimentaire ou ajoutez-le manuellement dans l'application.

Obtenez une estimation de l'empreinte carbonne de votre produit, une note environnementale et de multiples données concernant son impact environnemental.

Retrouvez tous vos produits scannés, triez-les, cumulez leurs empreintes carbone etc.
